Data Mining is the process of using pattern recognition on vast sets of data to find relationships and trends. This can be useful for comapanies who wish to maximise profits (as market research). It is also used to spot errors in sets of data, outliers produced after trawling may be scrutinised as erreoneous data.

Performance modelling is the process of producing simulations to model situations before implementing these situations into the real world. This is useful when real world analysis may be dangerous, expensive and inconvenient.

Pipelining is where the output of one instruction is the input for the next; it can be used to speed up tasks being carried out.

Instruction Pipelining is used in CPUs to speed up processor time.

RISC Processors

Pipelining is a standard feature in RISC processors: the processor works on different stages of the instructions at the same time, so more instructions can be completed in a shorter period of time. However, in practice RISC processors operate at more than one cycle per instruction. The processor might occassionaly stall a result of data dependencies and branch instructions.
